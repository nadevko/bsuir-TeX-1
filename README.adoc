== Description

This project aims to generalize in a more convenient and understandable form the
provisions of the CoS, gather all verbal recommendations from lecturers and
departmental requirements in one place, implement templates for all types of
documents required at BSUIR, and create a convenient LaTeX environment for them.

== Installation

1. Download the files from the release.
2. Install in $TEXMFHOME/xelatex
** bsuir-sectioning.sty
** bsuir.cls
3. Install in $TEXMFHOME/bibtex/bst
** ugost2003.bst
