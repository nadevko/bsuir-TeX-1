== Апісанне

Гэты праект імкнецца абагульніць у больш зручнай і зразумелай форме палажэнні
СТП, сабраць у адным месцы ўсе вусныя рэкамендацыі лектараў і патрабаванні
кафедраў, рэалізаваць шаблоны ўсіх тыпаў дакументаў, патрабаваных у БДУІР, і
стварыць для іх зручнае LaTeX-асяроддзе.

== Усталёўка

1. Спампуйце файлы з рэлізу.
2. Усталюйце ў $TEXMFHOME/xelatex
** bsuir-sectioning.sty
** bsuir.cls
3. Усталюйце ў $TEXMFHOME/bibtex/bst
** ugost2003.bst
