== Описание

Этот проект стремится обобщить в более удобной и понятной форме положения СТП,
собрать в одном месте все устные рекомендации лекторов и требования кафедр,
реализовать шаблоны всех типов документов, необходимых в БГУИР, и создать для
них удобное LaTeX-окружение.

== Установка

1. Скачайте файлы из релиза.
2. Установите в $TEXMFHOME/xelatex
** bsuir-sectioning.sty
** bsuir.cls
3. Установите в $TEXMFHOME/bibtex/bst
** ugost2003.bst
